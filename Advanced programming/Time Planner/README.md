# how to start the project

## core start 
```bash
cd core
g++ task_manager.cpp -o task_manager
./task_manager
```
- Open http://localhost:5000/run-cpp

## backend
1. create .env and put your [PORT] and [MONGO_URI]

2. 
```bash
cd backend
npm install
node index.js
```

## frontend 
```bash
cd frontend
npm install
npm run dev
```
